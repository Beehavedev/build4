Add a "Live Feed" section to the Agents page in the

Each entry looks like:
  ┌─────────────────────────────────┐
  │ 🤖 AlphaHunter  2 mins ago     │

  │ ADX 18.4 | RSI 52 | BB squeeze │

  │ Size $12.50 | SL $2,290        │
  │ "Strong uptrend, MACD cross"   │
  └─────────────────────────────────┘

To build this:

1. Add an AgentLog table to the DB schema:
   model AgentLog {
     id        String   @id @default(cuid())
     agentId   String
     agent     Agent    @relation(fields: [agentId], references: [id])
     action    String   // HOLD, OPEN_LONG, OPEN_SHORT, CLOSE
     pair      String
     price     Float?
     reason    String   // Claude's reasoning (first 100 chars)
     adx       Float?
     rsi       Float?
     score     Int?
     createdAt DateTime @default(now())
   }

2. In the trading agent runner, after every decision,
   save to AgentLog:
   await db.agentLog.create({
     data: {
       agentId: agent.id,
       action: decision.action,
       pair: selectedPair,
       price: currentPrice,
       reason: decision.reasoning?.slice(0, 120) ?? '',
       adx: indicators.adx,
       rsi: indicators.rsi14,
       score: decision.score
     }
   })

3. Add API endpoint GET /api/agents/:id/feed
   Returns last 20 AgentLog entries for that agent

4. Add API endpoint GET /api/me/feed  
   Returns last 20 entries across ALL user's agents

5. In the mini app Agents tab, add a "Live Feed" 
   section below the agent list that polls 
   /api/me/feed every 30 seconds and shows the 
   timeline entries styled as cards above.
