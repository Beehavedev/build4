1. Struct encoding — use ethers.AbiCoder explicitly

Don't pass the struct as a plain JS object. Encode it manually:

typescript
const abiCoder = ethers.AbiCoder.defaultAbiCoder();

const encodedParams = abiCoder.encode(
  ['tuple(bool isMint, uint256 amount, bool isExactIn, uint256 minOutOrMaxIn)'],
  [{
    isMint: true,
    amount: amountIn,
    isExactIn: true,
    minOutOrMaxIn: 0n
  }]
);
Then pass encodedParams as the params argument to swapSimple, not the raw object.
