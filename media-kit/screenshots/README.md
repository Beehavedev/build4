## Capture guidelines

- **Resolution:** at least 2x retina (1920px wide minimum for landscape, 1170px for mobile portrait)
- **Format:** PNG (lossless) for UI; JPG only if photo
- **Crop:** include just enough chrome (status bar, etc.) to anchor the context, no more
- **PII:** redact wallet addresses, usernames, real balances unless intentional
- **Naming:** `NN-short-description.png` — keep alphabetical order = recommended display order
